LR
Logistic Regression
Linear probabilistic classifier — the baseline benchmark
Structure
Maps input features to a probability via a linear combination followed by the sigmoid function: P(fraud) = σ(w·x + b). A single decision boundary (hyperplane) separates classes. Weights w are learned by maximizing log-likelihood (or equivalently minimizing binary cross-entropy), optionally with L1/L2 regularization.

RF
Random Forest
Ensemble of decorrelated decision trees via bagging
Structure
Builds B decision trees in parallel, each trained on a bootstrap sample (≈63% of data) and at each node considers only a random subset of √p features. Final prediction is majority vote (classification) or average (regression). Variance is reduced by averaging many low-correlation trees.

XGB
XGBoost
Gradient boosting with regularization and second-order optimization
Structure
Sequentially adds trees, each correcting the residual error of the previous ensemble. Uses second-order Taylor expansion of the loss (both gradient g and hessian h) to compute optimal leaf weights exactly: w* = −G / (H + λ). L1 (α) and L2 (λ) regularization on leaf weights. Supports column subsampling, row subsampling, and monotone constraints.

Cat
CatBoost
Gradient boosting with symmetric trees and ordered boosting
Structure
Builds symmetric (oblivious) trees where every node at the same depth uses the same split condition — creating a balanced tree that acts like a lookup table for fast inference. Uses ordered boosting (a permutation-based scheme) to avoid prediction shift (target leakage during training). Categorical features are handled natively via target statistics without requiring preprocessing.

LGB
LightGBM
Leaf-wise gradient boosting with histogram binning — fastest at scale
Structure
Uses leaf-wise (best-first) tree growth instead of level-wise: always splits the leaf with the highest loss reduction, producing deeper, more asymmetric trees. Gradient-based One-Side Sampling (GOSS) retains only high-gradient instances + random sample of low-gradient ones, drastically cutting data. Exclusive Feature Bundling (EFB) merges mutually exclusive sparse features, reducing dimensionality. Histogram binning quantizes continuous features.
